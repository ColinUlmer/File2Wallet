# File2Wallet
Command-line tool for generating Ethereum wallet private key and public address from any file of any type<br>
Requires: Node.js

<h2>Install</h2>

```
git clone https://github.com/ColinUlmer/File2Wallet.git
npm install -g
```

<h2>Usage</h2>

```
f2w -s <sha3Cycles> <path_to_file>
```
